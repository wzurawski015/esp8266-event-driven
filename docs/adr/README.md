# Architecture Decision Records

This directory contains Architecture Decision Records (ADRs) for the
**esp8266-event-driven** project.

ADRs document significant architectural choices: what was decided, why, and what
alternatives were rejected.

## Index

| Number | Title | Status |
|--------|-------|--------|
| [0000](0000-template.md) | ADR template | — |

## Creating a new ADR

1. Copy `0000-template.md` to `NNNN-short-title.md` where `NNNN` is the next
   available four-digit number.
2. Fill in all sections.
3. Set the **Status** to `Proposed`.
4. Submit a pull request. The ADR is accepted when the PR is merged.

## Statuses

| Status | Meaning |
|--------|---------|
| Proposed | Under discussion |
| Accepted | Decision adopted |
| Deprecated | No longer applies |
| Superseded by [NNNN] | Replaced by a later ADR |
